# cudup list

List available CUDA versions.

## Usage

```bash
cudup list
```

## Output

Shows all CUDA versions available for installation. Installed versions are marked with `*`.

```
Available CUDA versions:
*      12.4.1
*      12.4.0
       12.3.2
       12.3.1
       11.8.0
       11.7.1

* = installed
```
